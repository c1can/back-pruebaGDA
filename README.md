## Instalacion

Instala las dependencias con npm

```bash
  npm install
```

Crea un un archivo .env en la raiz del proyect

### archivo .env

```javascript
PORT=3000 || prefencia
DB_HOST=localhost
DB_NAME=nombre_base_de_datos
DB_PASSWORD=contrasena
```
### Levanta el proyecto

```bash
  node .\index.js
```
